### Installing

```

BASIC_AUTH_USERNAME=
BASIC_AUTH_PASSWORD=
PUBLIC_KEY_PATH=
PRIVATE_KEY_PATH=
DSN_SENTRY_URL=
MONGO_DATABASE_URL=
```
- Then run this command
```

...
$ npm install
$ npm start
```
